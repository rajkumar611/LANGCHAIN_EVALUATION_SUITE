NEXUS CORP — ACTIVE PROJECT REGISTRY
=====================================

This document lists all active engineering projects at Nexus Corp as of Q1 2025. Each project entry includes the owning team, key technologies in use, and current status.

PROJECT NOVA
------------
Status:       Active (In Development)
Owner:        Platform Team + AI Research Team (joint project)
Project Lead: Alex Kumar (Platform Team), Elena Vasquez (AI Research Team)
Description:  Real-time ML inference platform. Routes prediction requests to the appropriate model endpoint, with auto-scaling based on request volume.
Technologies in use:
  - Kubernetes v1.28      (Container orchestration for inference pods)
  - PyTorch v2.1          (Model serving via TorchServe)
  - Redis v7.2            (Request queue and result caching)
  - PostgreSQL v15.4      (Request logging and audit trail)

PROJECT SHIELD
