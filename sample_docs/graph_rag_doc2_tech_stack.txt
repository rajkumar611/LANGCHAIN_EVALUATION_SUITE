NEXUS CORP — TECHNOLOGY STACK REGISTRY
=======================================

This document records the approved technology stack in active use by each engineering team. It is maintained by the Platform Team and reviewed by the Security Team each quarter. Last updated: Q1 2025.

PLATFORM TEAM — APPROVED TECHNOLOGIES
--------------------------------------
Container Orchestration:
  - Kubernetes v1.28           (Production cluster — AWS EKS)
  - Docker v24.0               (Local development and CI builds)

Infrastructure as Code:
  - Terraform v1.6             (Cloud resource provisioning)
  - Ansible v2.15              (Configuration management)

Databases:
  - PostgreSQL v15.4           (Primary relational store)
  - Redis v7.2                 (Session cache and job queue)

Observability:
  - Prometheus v2.47           (Metrics collection)
  - Grafana v10.1              (Dashboarding and alerting)

AI RESEARCH TEAM — APPROVED TECHNOLOGIES
-----------------------------------------
ML Frameworks:
  - PyTorch v2.1               (Primary training framework)
  - Hugging Face Transformers v4.35  (Model hub and fine-tuning)

Experiment Tracking:
  - MLflow v2.8                (Experiment logging and model registry)

Compute:
  - Kubernetes v1.28           (Training job orchestration via KubeFlow)

Storage:
  - PostgreSQL v15.4           (Experiment metadata)
  - S3-compatible object store (Model artefact storage)

SECURITY TEAM — APPROVED TECHNOLOGIES
--------------------------------------
Secret Management:
  - HashiCorp Vault v1.15      (Secrets and certificate management)

Runtime Security:
  - Falco v0.36                (Container runtime threat detection)
  - Wazuh v4.6                 (SIEM and log analysis)

Vulnerability Scanning:
  - Trivy v0.47                (Container image scanning)
  - Snyk                       (Dependency vulnerability scanning)

NOTE: All teams must obtain Security Team approval before adding, upgrading, or retiring any technology listed above. Unapproved technology usage is a policy violation.
